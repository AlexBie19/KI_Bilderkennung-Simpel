# KI_Bilderkennung-Simpel – Clothing Recognizer

A Flutter app that uses the device camera to photograph a clothing item and
identifies its type (T-shirt, Hoodie, Jeans, …) with a neural network trained
on the [Fashion-MNIST](https://github.com/zalandoresearch/fashion-mnist) dataset.

## Quick start

### 1 · Train the model (required before running the app)

```bash
cd ai_model
pip install -r requirements.txt
python train_fashion_mnist.py
```

This trains the CNN on Fashion-MNIST and writes the TFLite model directly to
`flutter_app/assets/models/fashion_mnist_model.tflite`.

Optional flags:
```bash
python train_fashion_mnist.py --epochs 20 --output /custom/path/model.tflite
```

### 2 · Run the Flutter app

```bash
cd flutter_app
flutter pub get
flutter run
```

The app requires a physical device (or an emulator with camera support).

## How it works

1. **Camera capture** – `CameraScreen` requests the `CAMERA` permission (via
   `permission_handler`) and shows a live preview.  Tapping the button calls
   `CameraController.takePicture()`.

2. **Image preprocessing** – `ImagePreprocessingService.preprocessImageForFashionMnist()`
   converts the captured photo to 28 × 28 grayscale and normalises pixel values
   to `[0.0, 1.0]`, exactly matching the format used in Fashion-MNIST.

3. **Inference** – `ClothingClassifierService` loads `fashion_mnist_model.tflite`
   via `tflite_flutter`, reshapes the tensor to `[1, 28, 28, 1]`, runs the
   model, and picks the class with the highest softmax probability (argmax).

4. **Result display** – `ResultScreen` shows the captured image alongside a
   `ClassificationResultWidget` card with the label and a confidence bar.

## Extending the app

The project is intentionally structured so that new capabilities can be added
without touching existing files:

| Goal | Where to change |
|------|----------------|
| Add color detection | Create `ColorClassifierService` + a new TFLite model trained on RGB images |
| Add pattern detection | Create `PatternClassifierService` + a new model |
| Show all class probabilities | Add a `topKResults` getter to `ClassificationResult` and update `ClassificationResultWidget` |
| Improve accuracy | Increase `--epochs`, add data augmentation in `train_fashion_mnist.py` |

The commented-out extension hooks in `ImagePreprocessingService` and
`ClothingClassifierService` show exactly where to insert new logic.
